## 1.Caesar Cipher 

This project implements the Caesar Cipher encryption technique in Java.

 Features
- Encrypts text using a shift value
- Demonstrates string and character manipulation
- Beginner-friendly cryptography project


## Goldbach's Conjecture

A Java program that accepts an even number between 9 and 50 and displays all prime pairs whose sum equals the given number.
Features
- Input validation
- Methods
- Loops
- Prime number logic
- Mathematical problem solving
